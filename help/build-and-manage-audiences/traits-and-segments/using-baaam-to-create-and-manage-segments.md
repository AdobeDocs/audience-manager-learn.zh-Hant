# 使用BAAAM建立和管理區段

使用Audience Manager中的BAAAM工具大量建立和管理您的區段，將其放入資料夾、新增特徵至區段規則、對應至目的地等。

>[!IMPORTANT]
>
>BAAAM需要在64位元Microsoft Windows上執行的Microsoft Excel。 建議您使用最新版Microsoft Excel。 macOS上不再支援大量管理工具。
