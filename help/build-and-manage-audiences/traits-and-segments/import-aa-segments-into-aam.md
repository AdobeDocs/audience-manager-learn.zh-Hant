# 將區段從Adobe Analytics匯入Audience Manager

除了將即時資料從Adobe Analytics轉送至Audience Manager，您也可以匯入區段，其中包含從Analytics進行後處理的資料，並透過Experience Cloud將區段匯入Audience Manager。 這是其他DMP所無法誇耀的強大功能。
