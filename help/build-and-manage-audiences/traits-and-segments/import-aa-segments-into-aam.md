# 從Adobe Analytics將區段匯入Audience Manager

除了將即時資料從Adobe Analytics轉送至Audience Manager外，您也可以透過Experience Cloud將包含Analytics後處理資料的區段匯入Audience Manager。 這是其他DMP所無法誇耀的強大功能。
