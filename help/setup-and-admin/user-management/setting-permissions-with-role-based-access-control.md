# 使用基於角色的訪問控制設定權限

在Adobe Audience Manager，基於角色的訪問控制允許您在組級別管理權限，控制哪些人可以查看和使用資產，包括特徵、段、目標和模型。 瞭解如何設定權限組並向其中添加用戶。
