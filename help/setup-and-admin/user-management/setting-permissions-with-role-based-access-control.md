# 使用角色型存取控制設定許可權

Adobe Audience Manager中的角色型存取控制可讓您在群組層級管理許可權，控制哪些人可以檢視和使用特徵、區段、目的地和模型等資產。 瞭解如何設定許可權群組及將使用者新增至許可權群組。
